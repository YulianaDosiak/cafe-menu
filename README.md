# Cafe Menu

A stylish, digital cafe menu designed to showcase foundational CSS styling techniques alongside HTML structure.

## Technologies Used
* HTML5
* CSS3

## What I Learned
In this project, I focused on styling and design, learning how to:
* Link external CSS files to HTML documents.
* Use CSS selectors to target specific elements.
* Work with colors, fonts, margins, and padding to create a visually appealing layout.
